# SolariView

> **On-chain clarity, local-first security.**

SolariView is a high-fidelity CLI tool designed for the sovereign developer. It aggregates your multi-chain digital footprint—portfolios, NFT utility, and smart contract interactions—directly into your terminal without leaking metadata to centralized third-party APIs.

## 🚀 Features
* **Zero-Telemetry:** No tracking, no remote logging, no data leaks.
* **Multi-Chain Aggregation:** Unified view across EVM, L2s, and beyond.
* **Developer-First:** Clean JSON output for easy piping into other local tools.
* **Privacy-First:** Local state management ensures your addresses remain your business.

## 🛠 Installation

# Clone the repository
git clone https://github.com/DawnSolari/SolariView.git

# Install dependencies
cd SolariView
npm install 

## 📋 Usage

# Run the tool with a target address
solariview --address <0x...> --chain all

## 🛡 Philosophy
Watching your assets shouldn't mean being watched. SolariView is built for utility and community, prioritizing skill-based rewards and technical transparency over speculative structures.
